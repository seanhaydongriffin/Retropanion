Appendix A: SSH Setup


Retropanion requires a SSH connection to your RetroPie to work.

If you haven't setup a SSH connection yet please follow the instructions on the RetroPie website below ...

https://retropie.org.uk/docs/SSH

In my opinion the easiest method is (Option 2) From the RetroPie menu ...
  1. Select raspi-config from the RetroPie menu after booting up.
  2. Select Interface Options
  3. Select SSH
  4. Choose Yes
  5. Select Ok
  6. Choose Finish
Then use your chosen SSH client (ie. PuTTY or WinSCP) with the following connection details (assuming the default values) ...
  • Host Name: retropie
  • Port: 22
  • Username: pi
  • Password: raspberry
